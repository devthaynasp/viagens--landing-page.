# 🌍 Viagens - Landing Page

Esta é a solução para o desafio de projeto de uma Landing Page de viagens. O objetivo foi criar uma página moderna, responsiva e visualmente atraente utilizando tecnologias fundamentais do desenvolvimento web.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização, layout (Flexbox/Grid) e responsividade.
* **Google Fonts:** Tipografia personalizada para melhor legibilidade.
